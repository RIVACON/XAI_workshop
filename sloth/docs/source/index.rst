
Welcome to sloth's documentation!
==================================

Sloth is a framework for the model agnostic validation and interpretation of machine learning models 
on tabular data with the main focus on models from the financial domain. It implements algorithms and interfaces 
other libraries. In addition it provides out-of-the-box plots to present the results in a adequate fashion.
